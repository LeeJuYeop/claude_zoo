## 현재
플러그인과 서브에이전트만을 카드로 표시

## 목표
- 플러그인+MCP+서브에이전트+skills+commands+hooks 를 모두 보여주기
- 기존의 scope 별 탭은 유지. (서브에이전트 탭은 삭제). 각 scope에 적용된 생태계를 모두 보여주기
- 체크박스 필터링을 통해서 플러그인+MCP+서브에이전트+skills+commands+hooks 중에서 원하는 것만 볼 수 있게하기. default는 전체 다 보여주기.

## 세부디테일
- 서브에이전트+skills+commands+hooks는 카드 선택 시 표시 정보에 description 항목 추가.
