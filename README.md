# ClaudeZoo

로컬 머신에 설치된 Claude Code 플러그인 생태계를 시각화하는 데스크탑 앱.

플러그인, 에이전트, 스킬, MCP 서버, 훅을 한 눈에 확인할 수 있습니다.

## 요구사항

- [Node.js](https://nodejs.org/) 20 이상
- [Git](https://git-scm.com/)
- Claude Code가 설치된 환경 (`~/.claude/` 디렉토리 존재)

## 설치 및 실행

```bash
git clone https://github.com/LeeJuYeop/ClaudeZoo.git
cd ClaudeZoo
npm install
npm start
```

pnpm을 사용한다면:

```bash
pnpm install
pnpm start
```

## 업데이트

```bash
git pull
npm install
npm start
```
